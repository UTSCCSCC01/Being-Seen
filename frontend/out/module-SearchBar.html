Module: SearchBar

This components accepts a user input and navigates the user to a dedicated search result screen (SearchScreen) when the user submits.
Properties:
Name Type Attributes Description
navigation object The navigation object provided by the react navigation library.
resultScreenName string The name of the screen to which this component will navigate the user.
serviceType string The name of the api endpoint to which the search screen will send http requests.
prefill string <optional>
 The prefill text in the search bar. Default: ""
isSecondary bool <optional>
 Whether there will be a back button on the right side of the text input. Default: false
